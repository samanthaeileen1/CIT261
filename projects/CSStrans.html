CSS Transitions & Transforms

Here is an example of a simple transition. It transforms the width and height of the circle.
Note: We made the div into a circle by setting the border-radius property to 50%.
Note: It looks like an oval while it is transforming because we set the width to transform at the duration of 2 seconds, and the height to tranform at a duration of 4 seconds.



Here is a transform WITHOUT the transition. It transforms using the scale function in the transform property, like so:
transform: scale(2);
Since there is only one parameter, it will scale in the direction of the X AND the Y axis.



This does the same thing as the transfrom, but with a transition. Notice how much smoother it is.



Here is a transform WITHOUT the transition. It transforms using the scale function in the transform property, like so:
transform: scale(1,2);
