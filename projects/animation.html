CSS Animation

Here is an example of a simple animation. It uses the
from { background-color:white; }
to { background-color:teal; }
syntax. This makes it so you just have one trasition throughout the animation. It goes from the first set of CSS in the brackets to the second set.





Here is the second animation. It uses a different sytax to be able to do more tansitions during the set animation period. It works by percents, like so: 0% { background-color:teal; } 12% { background-color:teal; } 23% { background-color: #f442b3; } 37% { background-color: #9241f4; } 67% { background-color: #f9f570; border-radius: 50% } 80% { background-color: khaki; } 90% { background-color: #f99f70; } 100% { background-color: #fc7b37; } I set the duration to 10 seconds, so for this animation, it will spend the first 12% of the 10 seconds teal, then it will change at 23% of 10 seconds to the next color, and so on and so forth throughout the animation. You notice that you can change any sort of CSS property, like border-radius. You can really do whatever you want with it!
